**How to understand pratice**
1. Clone file advanced-sql-practice-mysql
2. Install docker
3. Fix .env 
4. Open cmd (Shift + right click on folder advanced-sql-practice-mysql) 
5. Docker pull mysql:8.0 (spam until successfull)
6. docker compose up -d
7. docker compose ps
8. docker compose exec mysql mysql -u root -p
9. open file excercise (.sql)
10. Let's start
